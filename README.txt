Encriptador y Desencriptador de Texto

Este proyecto es una aplicación web simple para cifrar y descifrar texto utilizando un algoritmo básico de sustitución. También incluye funcionalidades para copiar y pegar texto, así como limpiar el campo de entrada.

Características
Cifrado y Desencriptado: Utiliza un diccionario de sustitución para cifrar y descifrar texto.
Interfaz de Usuario: Dos áreas de texto para entrada y salida, con botones para cifrar, descifrar, copiar, pegar y limpiar.
Responsive Design: Se adapta a diferentes tamaños de pantalla.

Tecnologías Utilizadas
HTML: Estructura del contenido web.
CSS: Estilos y diseño.
JavaScript: Lógica de cifrado, descifrado, y funcionalidades de los botones.

Archivos del Proyecto
index.html: Estructura HTML de la aplicación.
styles.css: Hoja de estilos CSS para el diseño y la apariencia.
script.js: Archivo JavaScript que contiene la lógica para cifrar, descifrar y manejar los eventos de los botones.
Assets: carpeta con imágenes e íconos.
